# Prekshak H S - Developer Portfolio

Welcome to the repository for my personal developer portfolio! This project is a modern, responsive, and highly interactive web application designed to showcase my projects, professional experience, skills, and education.

## ✨ Features

- **Custom Neo-Brutalist Design**: A striking, artisanal aesthetic using sharp borders, bold typography (Fraunces & Work Sans), interactive 3D hover effects, and a warm color palette.
- **Interactive AI Chatbot**: A custom-built, frontend-powered AI assistant that can answer specific questions about my projects, skills, education, and how to contact me.
- **Interactive Background**: A lightweight, dynamically animated HTML5 Canvas background that responds to mouse movements.
- **Fully Responsive**: Carefully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Frontend**: Pure HTML5, CSS3 (Custom Variables & Flexbox/Grid), and Vanilla JavaScript (ES6+).
- **Icons & Typography**: Google Fonts (Space Mono, Work Sans).
- **Backend / Dev Server**: A lightweight Node.js/Express server is included (`server.js`) for local development, though the site itself is fully static and can run independently.

## 🚀 Getting Started

There are two ways to run this portfolio locally:

### 1. Static Mode (Simplest)
Since all logic (including the AI chatbot) is handled client-side, you can simply open the `index.html` file in your browser:
```bash
# Navigate to the public directory
cd public

# Open index.html in your default web browser
```

### 2. Local Node Server
If you prefer to run it through the provided Express server:
```bash
# Install dependencies
npm install

# Start the server
npm start
```
Then, visit `http://localhost:3000` in your browser.

## 📦 Deployment

This project is fully static and ready for production deployment. It can easily be deployed for free on platforms like **Vercel**, **Netlify**, or **GitHub Pages**. 
*If deploying on Vercel, simply import the repository and set the **Root Directory** to `public`.*
